# 배송 가이드

## 제1조 기본 배송 정보

1. 기본 배송비: 3,000원
2. 50,000원 이상 주문 시 무료배송
3. 배송사: CJ대한통운 (기본), 로젠택배 (도서산간 일부)
4. 영업일 기준 배송 처리 (토/일/공휴일 제외)

## 제2조 배송 소요 기간

1. 수도권 (서울/경기/인천): 결제 완료 후 1~2영업일
2. 광역시 및 기타 지역: 결제 완료 후 2~3영업일
3. 도서산간 지역 (제주, 울릉도 등): 결제 완료 후 3~5영업일
4. 주문 폭주 기간 (명절, 대규모 프로모션): 평시 대비 1~2영업일 추가 소요

## 제3조 도서산간 배송비

1. 도서산간 지역은 추가 배송비 3,000원이 부과됩니다.
2. VVIP 등급 고객은 도서산간 추가 배송비가 면제됩니다.
3. VIP 등급 고객은 70,000원 이상 주문 시 도서산간 추가 배송비가 면제됩니다.
4. 일반 등급 고객은 도서산간 추가 배송비 면제 혜택이 없으나, 50,000원 이상 주문 시 기본 배송비(3,000원)는 면제됩니다 (추가 배송비 3,000원은 별도 부과).

## 제4조 배송 지연 기준 및 보상

1. 배송 지연 기준: 결제 완료 후 3영업일(수도권), 4영업일(기타 지역), 6영업일(도서산간)이 경과하도록 '배송완료'되지 않은 경우
2. 배송 지연 확인 시 등급별 보상:
   - VVIP: 주문 금액의 10% 할인쿠폰 (최대 30,000원)
   - VIP: 적립금 5,000원
   - 일반: 적립금 2,000원
3. 보상은 배송 완료 후 자동 지급되며, 고객이 별도로 신청할 필요 없습니다.
4. 천재지변, 택배사 파업 등 불가항력 사유에 의한 지연은 보상 대상에서 제외됩니다.

## 제5조 배송 상태 안내

1. 주문확인: 주문 접수 완료, 상품 준비 중
2. 배송준비: 상품 포장 완료, 택배사 인계 대기
3. 배송중: 택배사에 인계 완료, 배송 진행 중
4. 배송완료: 고객 수령 확인
5. 각 단계별 상태 변경 시 SMS/앱 푸시 알림이 발송됩니다.

## 제6조 수령 확인 및 자동 구매확정

1. 배송완료 후 7일 이내에 고객이 수령 확인하지 않으면 자동 구매확정 처리됩니다.
2. 자동 구매확정 시 구매 적립금이 지급됩니다 (결제 금액의 1%, VIP 2%, VVIP 3%).
3. 자동 구매확정 이후에도 반품 기한(7일) 내라면 반품 신청이 가능합니다.
