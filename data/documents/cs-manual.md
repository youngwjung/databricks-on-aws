# CS 대응 매뉴얼

## 제1조 불량 상품 교환 프로세스

1. 고객이 불량 신고 접수 (사진 증빙 필수)
2. 접수 후 1영업일 이내 택배사 수거 요청
3. 수거 완료 후 물류센터 검수 (1영업일)
4. 검수 확인 후 정상 상품 재발송 (1~2영업일)
5. 전체 프로세스 소요 기간: 수거 시작 후 3~5영업일
6. 불량 교환의 경우 배송비는 전액 판매자 부담

## 제2조 오배송 처리 절차

1. 고객이 오배송 신고 (수령 상품 사진 증빙)
2. CS팀에서 주문 내역과 대조하여 오배송 확인 (당일 처리)
3. 확인 즉시 정상 상품 선발송 (수거 대기 없음)
4. 오배송 상품 택배 수거 요청
5. 3영업일 내 미수거 시 고객 임의 처분 가능 안내
6. 사과 적립금 2,000원 자동 지급

## 제3조 배송 지연 클레임 대응

1. 고객 문의 접수 시 즉시 배송 상태 확인
2. 택배사 추적 시스템에서 현재 위치 및 사유 파악
3. 지연 기준 초과 확인 시:
   - 고객에게 예상 도착일 안내
   - 등급별 보상 안내 (배송 가이드 제4조 참조)
   - 보상은 배송 완료 후 자동 지급됨을 안내
4. 고객이 주문 취소를 원하는 경우:
   - '배송중' 상태라도 배송 지연 사유 시 취소 가능 (예외 처리)
   - 취소 승인 후 택배 회수 요청

## 제4조 에스컬레이션 기준

1. Level 1 (일반 상담원 처리):
   - 단순 배송 조회, 주문 상태 안내
   - 반품/교환 접수, 쿠폰/포인트 안내
2. Level 2 (선임 상담원 처리):
   - 환불 지연 7영업일 초과 건
   - 동일 건 3회 이상 재문의
   - 고객 불만 표현 (욕설, 강경 어조)
3. Level 3 (매니저 처리):
   - 법적 조치 언급 건
   - VVIP 고객 불만 건
   - 언론/SNS 게시 언급 건
   - 동일 건 5회 이상 재문의

## 제5조 VVIP 고객 응대 가이드

1. VVIP 고객 인입 시 전담 CS 우선 연결 (대기시간 0분 목표)
2. VVIP 혜택 전체 목록:
   - 전 상품 무료배송 (도서산간 포함)
   - 반품 배송비 무료 (단순변심 포함, 연간 무제한)
   - 배송 지연 보상: 주문 금액의 10% 할인쿠폰 (최대 30,000원)
   - 전용 쿠폰 월 1회 발급 (15% 할인, 최대 15,000원)
   - 구매 적립률 3%
   - 생일 월 더블 적립 (6%)
3. VVIP 고객 이탈 방지:
   - 불만 표현 시 즉시 Level 3 에스컬레이션
   - 사과 쿠폰 상한 30,000원까지 자율 발급 권한
   - 3개월 미구매 시 리텐션 쿠폰 자동 발송

## 제6조 자주 묻는 질문 (FAQ) 응대

### 주문/결제
- "주문 취소하고 싶어요" → 주문 상태 확인, '배송준비' 전이면 즉시 취소 처리
- "결제가 두 번 됐어요" → 카드사 임시 승인 확인, 중복 결제 맞으면 즉시 취소

### 배송
- "언제 오나요?" → 주문 상태 + 지역별 예상 소요일 안내
- "택배 분실된 것 같아요" → 배송완료인데 미수령 시 택배사 조사 접수 (3영업일 소요)

### 반품/환불
- "환불 언제 되나요?" → 반품 상태 + 결제수단별 소요 기간 안내
- "포장 뜯었는데 반품 되나요?" → 상품 종류별 확인 (식품/속옷 불가, 전자기기 가능)

### 포인트/쿠폰
- "포인트 사라졌어요" → 유효기간(12개월) 만료 여부 확인
- "쿠폰이 적용 안 돼요" → 최소 주문금액, 유효기간, 쿠폰+포인트 중복 불가 안내
